# Business Card Organizer AI

A mobile-friendly web application that uses AI to organize business cards.

## Features
- **AI-Powered**: Extract contact info from photos using Gemini AI.
- **Multi-Card Support**: Recognizes multiple cards in a single photo.
- **Smart Grouping**: Automatically groups people by Company.
- **Privacy First**: Your API Key is stored locally in your browser.

## How to use on iPhone
1. Open the website.
2. Tap "Share" -> "Add to Home Screen".
3. Use it like a native app!

## Setup
You need a [Google Gemini API Key](https://aistudio.google.com/app/apikey) to use the AI features.
